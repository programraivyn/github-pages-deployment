Building and Deploying a Cheat Sheet

Git commands:

• git init:
creates a new Git repository.
• git clone:
creates a copy of an existing Git repository.
• git add:
updates the index using the current content found in the working tree.
• git commit:
performs a commit; adding -m "message" adds a message to the commit.
• git push:
uploads local repository content to a remote repository.

Bash commands:

• cd:
displays the current drive and directory.
• ls:
lists all the files in the current directory except for hidden ones.
• touch:
creates a new file or updates the timestamp and modification times to an existing file.
• mkdir:
makes a directory also known as a folder.
• pwd:
print working directory; prints the path of the working directory from the root.
